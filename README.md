# Disandu Perera Portfolio

A modern, interactive portfolio website showcasing my skills, projects, and experience as a Full Stack Developer and UI/UX Designer. Built with HTML, CSS, JavaScript, and Three.js for an immersive 3D experience.

## 🌟 Features

- **Interactive 3D Backgrounds**: Powered by Three.js for dynamic visual effects
- **Responsive Design**: Optimized for desktop and mobile devices
- **Skills Showcase**: Detailed breakdown of frontend, backend, and tool proficiencies
- **Project Gallery**: Comprehensive collection of personal projects including web apps, games, and ML models
- **Contact Form**: Integrated email functionality using EmailJS
- **Download CV**: Direct link to resume download
- **Smooth Animations**: CSS and JavaScript animations for enhanced user experience

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3 (Custom styles + Tailwind CSS)
- JavaScript (ES6+)
- Three.js (3D graphics library)

### Libraries & Frameworks
- Tailwind CSS (Utility-first CSS framework)
- Font Awesome (Icons)
- SweetAlert2 (Alerts and modals)
- EmailJS (Email service)

### Backend Integration
- EmailJS for contact form functionality

## 🚀 Installation & Usage

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection (for CDN resources)

### Running Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/BinaryVortex/Disandu-Perera-Portfolio.git
   cd Disandu-Perera-Portfolio
   ```

2. Open `index.html` in your web browser:
   - Double-click the file, or
   - Right-click and select "Open with" your preferred browser

The website will load and function locally, as all assets are included or loaded via CDN.
